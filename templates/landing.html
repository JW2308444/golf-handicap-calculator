⛳ Golf Calculator

This web application allows golfers to track their performance, record round data, and calculate a dynamic handicap index based on official scoring principles. It provides visual insights into score trends, identifies performance improvements over time, and helps users better understand how their handicap is derived. The system is designed to be simple, responsive, and accessible across all devices, ensuring a smooth user experience whether on desktop, tablet, or mobile.
